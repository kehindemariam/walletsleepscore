# walletsleepscore

A **wallet sleep-score** for the [Pharos Network](https://pharos.xyz). Point it at any wallet on Pharos Atlantic Testnet or Pacific Ocean Mainnet and you get a 0-100 "sleep hygiene" score plus a per-metric breakdown — recency, frequency, gas efficiency, dust ratio, interaction diversity, contract exposure, and balance activity. No third-party APIs, no portfolio lookups, no PII — just on-chain behavior scored against 7 transparent metrics.

Built for two use cases: triaging a list of wallets (which ones are bots, which are real users), and giving agents a single numeric handle for "is this address worth talking to?" without running a full audit.

Drop `SKILL.md` into your agent's skills directory and the agent can score any Pharos wallet on demand. Works with Claude Code, Codex, OpenClaw, and the Pharos Agent Center.

## What it scores

| # | Metric | Weight | What it measures |
|---|---|---:|---|
| 1 | **Recency** | 25% | Days since last outbound tx |
| 2 | **Frequency** | 15% | Outbound txs per week, lifetime average |
| 3 | **Gas efficiency** | 10% | Median gas-price percentile vs network median |
| 4 | **Dust ratio** | 10% | % of UTXOs that are dust |
| 5 | **Interaction diversity** | 15% | Distinct counterparties per month |
| 6 | **Contract exposure** | 10% | % of txs that interact with verified contracts |
| 7 | **Balance activity** | 15% | Lifetime gas-spent as % of current balance |

The overall score is a **weighted geometric mean** of the 7 metric scores.

## Score labels

| Score | Label |
|---:|---|
| 90-100 | **WIDE AWAKE** |
| 70-89 | **HEALTHY** |
| 50-69 | **DROWSY** |
| 30-49 | **LIGHT SLEEPER** |
| 10-29 | **DEEP SLEEPER** |
| 0-9 | **COMATOSE** |

## Install

```bash
# Option A — git clone
git clone https://github.com/kehindemariam/walletsleepscore.git
cd walletsleepscore
chmod +x scripts/score.sh scripts/score_demo.sh
pip install web3            # only for the Python version

# Option B — one-line via OpenClaw
npx skills add https://github.com/kehindemariam/walletsleepscore

# Or: install as a Pharos Agent Center / Claude Code / Codex / OpenClaw skill
mkdir -p ~/.pharos/skills
cp -r . ~/.pharos/skills/walletsleepscore
```

## Quick start

### Zero-dependency (bash + curl only)

```bash
# Default: Markdown report, mainnet
bash scripts/score.sh 0xWALLET --network mainnet

# Machine-readable JSON
bash scripts/score.sh 0xWALLET --network mainnet --format json

# Bounded scan for very old wallets
bash scripts/score.sh 0xWALLET --network mainnet --max-blocks 5000 --txs 200
```

### Python (richer output)

```bash
pip install web3
python3 scripts/score.py 0xWALLET --network mainnet --format md
```

### Run the demo

```bash
bash scripts/score_demo.sh
```

Scores a real public mainnet wallet and prints a sample report.

### Verify the install

```bash
bash scripts/score.sh --help
python3 scripts/score.py --help
python3 tests/test_score.py
```

## Honest scope

walletsleepscore is a **transparent, interpretable** behavioral score. It does NOT use ML, does NOT make sybil-vs-legit calls, does NOT pull third-party data. Use it to triage wallets, not to ban them.
